# AGENTS.md — Правила для AI-ассистента

## Стандартные правила проекта

- Используем **TypeScript** с ESM (`"type": "module"` в package.json).
- Шаблоны — **EJS**, клиентский JS — **Vanilla JS** (без фреймворков).
- Все данные хранятся **in-memory** (Map в `src/data/store.ts`), без БД.
- При импортах в `.ts` файлах обязательно указывать расширение `.js`.
- Роуты: admin → `/admin/*`, public → `/` и `/book/*`.
- Код без комментариев, если не сказано иного.
- Следовать структуре из PLAN.md.

## Поиск в node_modules

Не искать и не просматривать файлы в `node_modules/`, если не было явной команды проверить или установить зависимость.

## Данные из головы

Если во время работы нужно придумать название, описание, текст или любые другие данные, которых нет в PLAN.md, spec-файлах или коде проекта — **сначала спросить у пользователя**, а не брать из головы.
